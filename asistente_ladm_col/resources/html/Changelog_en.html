Cambios en la version 3.1.11

  • Reglas de calidad:
    • Novedades en reglas de calidad para IGAC:
      • 1 nueva regla de calidad (4023) para unidades de construcción (8 tipos de errores).
      • 9 nuevos tipo de error para la regla de calidad 4007 (interesados naturales): el campo 'sexo' no puede ser nulo; no se permiten números ni caracteres especiales para nombres y apellidos.
      • 2 nuevos tipos de error para la regla de calidad 4008 (interesados jurídicos): el campo 'sexo' debe ser nulo; el segundo nombre y el segundo apellido deben ser nulos.
    • Si existen geometrías de entrada no válidas (según la definición del OGC, y por tanto de GEOS), el Asistente LADM-COL las arregla antes de ejecutar una regla de calidad topológica. Los datos fuente no son alterados.
    • Arreglo para reglas de consistencia lógica en GeoPackage (usando GLOB en lugar de REGEXP para evaluar expresiones regulares).
