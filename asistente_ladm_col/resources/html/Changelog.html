Cambios en la version 4.0.1-beta

  • Se actualiza la versión del modelo de Levantamiento Catastral v1.2.

  • Se migran las siguientes reglas de calidad:

    • Reglas de calidad de puntos:

      • Los puntos de control no deben superponerse
      • Los Puntos de Lindero deben estar cubiertos por nodos de Lindero
      • Los puntos de lindero deben estar cubiertos por nodos de terreno
      • Los nodos de terreno deben estar cubiertos por puntos de lindero

    • Reglas de calidad de líneas

      • Los límites de Terreno deben estar cubiertos por Linderos

    • Reglas de calidad de polígonos:

      • Las construcciones no deben superponerse
      • Los servidumbres de tránsito no deben superponerse
      • Los servidumbres de tránsito no deben superponerse con construcciones
      • Las servidumbres de tránsito no deben tener geometrías multiparte

    • Reglas de calidad de consistencia lógica:

      • Los porcentajes de participación de la agrupación de interesados deben sumar uno
      • El código de departamento de la tabla predio debe tener dos caracteres numéricos
      • El código de municipio asociado al predio debe ser valido
      • Revisar que el número predial tiene 30 caracteres numéricos
      • El número predial anterior debe tener 20 caracteres numéricos
      • Interesado natural no puede incluir datos de interesado jurídico
      • Interesado jurídico no puede incluir datos de interesado natural
      • Punto Lindero no debe tener registros repetidos
      • Punto de Levantamiento no debe tener registros duplicados
      • Punto control no debe tener registros repetidos
      • Lindero no debe tener registros duplicados
      • Construcción no debe tener registros repetidos
      • Unidad de construcción no debe tener registros duplicados
      • Interesado no debe tener registros repetidos
      • Derecho no debe tener registros duplicados
      • Restricción no debe tener registros repetidos
      • Fuente administrativa no debe tener registros duplicados

  • El orden de los atributos de cada formulario se actualizo para una mejor su visualización.
